üì¢ How to Update Announcements

Announcements are stored in a simple text file that you can edit. No coding knowledge required!

1Find the Announcements File

Navigate to: new-design/data/announcements.json

You can open this file with any text editor (Notepad, TextEdit, etc.)

2Understanding the File Structure

The file looks like this:

{
  "announcements": [
    {
      "id": 1,
      "title": "Welcome to St. Patrick's Catholic Church",
      "date": "2025-01-15",
      "category": "General",
      "message": "We welcome all parishioners and visitors to our parish.",
      "priority": "normal"
    }
  ]
}

Each announcement has:

  • id: A unique number (1, 2, 3...)
  • title: The announcement headline
  • date: Format: YYYY-MM-DD (e.g., 2025-01-15)
  • category: Choose from: General, Liturgy, Information, Event, Important, Community, Education
  • message: The full announcement text
  • priority: Either "normal" or "high" (high shows with red badge)

3Adding a New Announcement

Copy an existing announcement block and paste it below the last one. Remember to:

  • Add a comma after the previous announcement's closing }
  • Give it a new unique ID number
  • Update the title, date, category, message, and priority

Example:

{
  "announcements": [
    {
      "id": 1,
      "title": "First Announcement",
      "date": "2025-01-15",
      "category": "General",
      "message": "This is the first announcement.",
      "priority": "normal"
    },
    {
      "id": 2,
      "title": "New Announcement",
      "date": "2025-01-20",
      "category": "Event",
      "message": "This is a new announcement about an upcoming event.",
      "priority": "high"
    }
  ]
}

‚ö†Ô∏è Important: Be careful with commas and quotation marks. Every opening { needs a closing }, and every opening quote " needs a closing quote ". If you make a mistake, the announcements won't display.

4Removing Old Announcements

Simply delete the entire announcement block (from { to }). Make sure to remove any extra commas.

üí° Tip: Keep a backup copy of the file before making changes. If something breaks, you can restore the backup.

üìÖ How to Manage Events (Google Calendar)

The Events page displays a Google Calendar. You'll need to set this up once, then just add events to your Google Calendar as normal.

1Create a Google Calendar

  1. Go to calendar.google.com
  2. Sign in with your church's Google account (or create one)
  3. Click the + next to "Other calendars"
  4. Select "Create new calendar"
  5. Name it "St. Patrick's Events" and click Create Calendar

2Make the Calendar Public

  1. In Google Calendar, find your "St. Patrick's Events" calendar in the left sidebar
  2. Click the three dots ‚ãÆ next to it
  3. Select "Settings and sharing"
  4. Scroll to "Access permissions"
  5. Check the box: "Make available to public"
  6. Click OK to confirm

3Get the Embed Code

  1. Still in Settings, scroll down to "Integrate calendar"
  2. You'll see an "Embed code" section with HTML code
  3. Click "Customize" to adjust the appearance (optional)
  4. Copy the entire <iframe>...</iframe> code

4Add to Your Website

  1. Open the file: new-design/upcoming-events.html
  2. Find the section marked "GOOGLE CALENDAR EMBED" (around line 140)
  3. Paste your iframe code where indicated
  4. Save the file

‚úÖ Done! Now whenever you add an event to your Google Calendar, it will automatically appear on the website. No need to edit the website again!

üì∏ How to Update Photo Gallery (Google Photos)

The easiest way to manage photos is using Google Photos albums.

1Create a Google Photos Album

  1. Go to photos.google.com
  2. Sign in with your church's Google account
  3. Click "Albums" in the left sidebar
  4. Click "Create album"
  5. Name it "St. Patrick's Parish Life"
  6. Add photos by clicking "Add photos"

2Share the Album Publicly

  1. Open your album
  2. Click the Share button (top right)
  3. Click "Create link"
  4. Copy the shareable link

3Two Options for Display

Option A: Direct Link (Easiest)

  1. Open: new-design/photo-gallery.html
  2. Find the "View Full Album" button section
  3. Replace the href="YOUR_GOOGLE_PHOTOS_LINK" with your actual link
  4. Users will click the button to view photos on Google Photos

Option B: Embedded Gallery (Advanced)

Some third-party tools allow embedding Google Photos. Search for "embed Google Photos album" for current options. This requires technical setup.

üí° Tip: You can add photos to the Google Photos album anytime from your phone or computer. They'll automatically appear when people click the gallery link.

üÜò Troubleshooting & Help

Problem: Announcements Not Showing

Solution:

  • Check the announcements.json file for missing commas or quotes
  • Make sure the date format is YYYY-MM-DD (e.g., 2025-01-15)
  • Restore from your backup copy if needed
  • Test the JSON syntax at jsonlint.com

Problem: Google Calendar Not Displaying

Solution:

  • Make sure the calendar is set to "Public"
  • Check that you copied the entire <iframe> code
  • Try generating a new embed code from Google Calendar

Problem: Photos Not Loading

Solution:

  • Verify the Google Photos album is set to "Public" or "Anyone with the link"
  • Make sure you copied the correct shareable link
  • Check that photos are actually in the album

Problem: Website Not Updating

Solution:

  • Clear your browser cache (Ctrl+Shift+Delete on Windows, Cmd+Shift+Delete on Mac)
  • Try viewing in a private/incognito window
  • Make sure you saved the file after editing
  • Make sure the file is uploaded to your web hosting (if applicable)
